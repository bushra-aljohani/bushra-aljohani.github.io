  1. Information
    Enhancing Medical Question Answering with LLMs via a Hybrid Retrieval-Augmented Generation Framework
    Bushra Aljohani and Tawfeeq Alsanoosy
    Information, Feb 2026
    Abs DOI Bib

    Given the knowledge-intensive and rapidly expanding nature of the medical field, accurately synthesizing and interpreting findings remain a major challenge for clinicians and medical students. Although Large Language Models (LLMs) have advanced automated summarization and response generation, their deployment is limited by hallucinations, outdated knowledge, and insufficient domain adaptation. Retrieval-Augmented Generation (RAG) addresses these issues by grounding LLMs in external knowledge bases. We examine a modular RAG framework with a hybrid retrieval strategy combining sparse retrieval (BM25) and dense retrieval (MedCPT). Evaluation on PubMedQA, MedMCQA, and MedQA-US using GPT-4o and BioGPT showed the hybrid retriever achieved 92.14% recall, 74.36% precision, and an F1-score of 82.30%. GPT-4o with hybrid retrieval reached 89.4% faithfulness and 82.7% answer relevancy on PubMedQA.

  2. PeerJ CS
    Pioneering Agentic Retrieval-Augmented Generation in Software Quality: A Novel Framework for Code Smell Detection via Dynamic Retrieval
    Bushra Aljohani and Abdulmajeed Aljuhani
    PeerJ Computer Science, Mar 2026
    Abs DOI Bib

    Code smells are subtle indicators of poor design choices that pose significant challenges to software maintainability and readability, particularly in dynamic languages such as Python. Traditional detection methods, including rule-based heuristics and static machine learning classifiers, often suffer from limited adaptability and lack of explainability. This study introduces a novel Agentic Retrieval-Augmented Generation (Agentic RAG) framework for code smell detection, marking the first application of agentic reasoning in this domain. The framework dynamically routes queries, selects optimal retrieval strategies, and synthesizes context-aware explanations using LLMs. Agentic RAG achieves 89.5% accuracy, a macro F1-score of 78.3%, and a weighted F1 of 88.7%.
